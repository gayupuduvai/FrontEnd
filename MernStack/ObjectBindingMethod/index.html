Object Binding Methods

1. call(): It is an Object binding, which is called onspot. And comma is used as seperator in it's Arguments.


2. apply(): It can be called out for future or lateron process. Arguments are given like an array.


3. bind(): It will bind an object and a function, and it as an Function


4. this: In Global scope and inside a function this() refers to Gobal Object Window. In Strict mode, when we use any function then this() returns as undefined. In Arrow function and in IIF it goes as Global Object Window. Mostly it will go with Anonymous Function.
